# BMI Calculator (Python)

## Project Overview
This project is a command-line based Body Mass Index (BMI) Calculator
developed using Python. It allows users to enter their weight and height,
calculates the BMI value, and classifies it into standard health categories.

## Features
- User-friendly command-line interface
- Input validation for weight and height
- Accurate BMI calculation
- Health status classification
- Option to perform multiple calculations

## Technologies Used
- Python 3

## How BMI is Calculated
BMI = weight (kg) / (height in meters)^2
