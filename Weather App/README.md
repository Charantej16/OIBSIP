# Weather Application (GUI)

This project is a real-time Weather Application developed using Python and Tkinter.
It fetches live weather data from the OpenWeather API and displays it in a clean graphical interface.

## Features
- Real-time weather information
- User-friendly GUI
- City-based search
- Error handling for invalid inputs and network issues

## Technologies Used
- Python
- Tkinter
- OpenWeather API

## How to Run
1. Install Python
2. Install required library:
   pip install requests
3. Add your OpenWeather API key in the code
4. Run:
   python weather_gui.py

## Note
An active internet connection is required to fetch live weather data.
