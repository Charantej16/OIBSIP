# Command Line Weather Application

## Project Overview
This project is a Python-based command-line application that retrieves
real-time weather information for a given city using a public weather API.

## Features
- Accepts city name from user
- Displays temperature, humidity, and weather condition
- Handles invalid input and network errors gracefully

## Tools & Technologies
- Python
- Requests Library
- OpenWeatherMap API

## Execution Steps
1. Install Python
2. Install requests library using pip
3. Add your API key inside the source file
4. Run the script using:
   python weather_app.py
