# Multi Client Chat Application

This project is a real-time multi-client chat application developed using Python.
It follows a client-server architecture where multiple users can communicate
simultaneously through a graphical interface.

## Technologies Used
- Python
- Socket Programming
- Threading
- Tkinter (GUI)

## Features
- Multiple clients can join the chat
- Real-time message broadcasting
- Username-based messaging
- GUI-based chat interface

## How to Run
1. Run server.py to start the server
2. Run client.py on multiple terminals
3. Enter username and start chatting

## Note
This application runs on localhost for educational purposes.
