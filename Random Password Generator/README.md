# Random Password Generator (GUI)

This project is a Python-based graphical application that generates secure
random passwords based on user-selected criteria.

## Features
- Graphical User Interface using Tkinter
- User-defined password length
- Options to include letters, numbers, and symbols
- Copy generated password to clipboard
- Simple, clean, and beginner-friendly design

## Technologies Used
- Python
- Tkinter

## How to Run the Project
1. Make sure Python is installed on your system
2. Open terminal or command prompt
3. Navigate to the project folder
4. Run the command:

   python random_password_gui.py

## Project Use Case
This application helps users generate strong and secure passwords
for better security in real-world applications.
